## 矩阵的计算


设存在2个矩阵 **A** 和 **B**
### 矩阵的加法
> 当2个矩阵为同型矩阵时，才可以进行加法运算
> 矩阵的加法满足交换律
> 矩阵和副矩阵的和为零矩阵
> **A** + **B** =  **B** + **A**
> ( **A** + **B** ) + **C**  =  **B** + ( **A** + **c** )

## 数与矩阵的乘法
> (λμ)**A** = λ(μ**A**)
> (λ+μ)**A** = λ**A** + μ**A**
> λ( **A** + **B** ) = λ**A** + λ**B**

### 矩阵的加法和乘法统称为矩阵的线性运算
